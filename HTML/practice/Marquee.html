Marquee Tag

  • Used to scrolling text or image
  • By default it goes to left
  • The marquee tag consist of 10 attributes :-
    • Direction = left,Right,up,down
    • Behavior = scroll,slide,alternate
    • hspace = horizontal space
    • vspace = vertical space
    • hight
    • width
    • bgcolor = red, rgb(125,45,66),#ccccc[Hexa decimal]
    • loop = how many times scroll
    • scroll amount = for increase speed
    • scroll delay = for decrese speed
    • NOTE :- Scrollamount & scrolldelay both are not used at sometime. At a time we can use only one attribute.
